


# React webpack2 boilerplate
  - webpack v2
  - react-hot-loader v3
  - react-router v4
  - assets pipeline (sass/postcss/autoprefixer)
  - code splitting, vendor and client bundles
  - lazy loading (via import())
  - source maps
  - basic eslint config
  - extract text plugin and manifest.json

```sh
$ yarn install

$ yarn start

$ yarn build
```
